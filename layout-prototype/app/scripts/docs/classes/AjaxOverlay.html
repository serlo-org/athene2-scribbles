AjaxOverlay Class

Defined in: libs/serlo/jquery.serlo_ajax_overlay.js:3

$.SerloAjaxOverlay opens links in an overlay. Creates an AjaxPage for every asynchronous loaded content, caches them and provides tabs

Available Callbacks: contentLoaded, contentOpened, beforeClose, beforeOpen, afterClose, beforeClose

Usage: $.SerloAjaxOverlay({on: {contentLoaded: function () { ... }}});
